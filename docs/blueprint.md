# **App Name**: SIP Tracker

## Core Features:

- Fund Tracking: Track the SIP performance of SUNDARAM MULTI-FACTOR FUND - REGULAR GROWTH, Motilal Oswal Innovation Opportunities Fund - Regular Plan Growth, UTI Mid Cap Fund Regular Plan Growth, ICICI Prudential Value Fund (erstwhile Value Discovery Fund) - Growth, and Bajaj Finserv Mutual Fund using the provided data.
- Data Import: Automatically import and update fund data from external URLs.
- Performance Visualization: Visualize the performance of each SIP using interactive charts and graphs.
- SIP Goal Setting: Allow users to set financial goals for their SIP investments and track progress.
- Personalized Insights: AI powered insights using the data to help user with recommendations.

## Style Guidelines:

- Primary color: Deep blue (#3F51B5) to evoke trust and stability, reflecting the serious nature of financial tracking.
- Background color: Light gray (#ECEFF1) for a clean, professional look that doesn't distract from the data.
- Accent color: Teal (#009688) to highlight key performance indicators and interactive elements.
- Body and headline font: 'Inter' sans-serif font for readability and a modern feel.
- Code font: 'Source Code Pro' for displaying code snippets.
- Use clear and simple icons to represent different funds and metrics.
- A clean and organized layout with intuitive navigation for easy access to fund information.
- Subtle transitions and animations to provide feedback on user interactions.
